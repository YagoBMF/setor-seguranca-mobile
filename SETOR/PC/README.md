# Setor Segurança — PC

Arquivos usados pelo atualizador automático da versão para PC.

## Estrutura

- `SETOR_SEG.lua`: mod completo para MoonLoader PC.
- `versao.txt`: versão mais recente disponível.

## Publicar uma atualização

1. Atualize o campo `versao` dentro de `SETOR_SEG.lua`.
2. Substitua `SETOR_SEG.lua` nesta pasta.
3. Coloque o mesmo número em `versao.txt`.
4. Envie os dois arquivos ao GitHub.

O jogador pode consultar com `/setorversao` e instalar com `/setoratualizar`.
